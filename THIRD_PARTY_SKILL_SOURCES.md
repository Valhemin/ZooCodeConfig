# Third-party Skill Sources and Inspiration

This bundle uses original, rewritten Zoo-compatible skill instructions inspired by several public skill/design resources discussed during setup:

- andrej-karpathy-skills style coding discipline: think before editing, keep changes simple, verify honestly.
- taste-skill style frontend taste layer: avoid generic AI-looking UI and improve visual craft.
- ui-ux-pro-max-skill style UX coverage: states, flows, accessibility, product reasoning.
- google-labs-code/design.md idea: keep a durable design-system source of truth in `DESIGN.md`.

No external repository content is vendored verbatim here. Review third-party licenses before directly copying their files into your own setup.
