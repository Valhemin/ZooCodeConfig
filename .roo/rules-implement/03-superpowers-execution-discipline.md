# Superpowers Execution Discipline

Implement from a written plan or create a short local plan before editing.

Default loop:
1. Pick the next coherent slice.
2. Confirm target files and expected behavior.
3. If tests exist or are practical, use RED-GREEN-REFACTOR.
4. Implement the smallest safe change.
5. Self-review against spec compliance and code quality.
6. Run focused verification.
7. Mark tasks done only after real completion.

Do not batch large unreviewed changes. Do not claim completion without verification evidence.
