# Production Readiness Gate

Before claiming completion, verify as much as practical.

Final status must include:
- What was completed.
- What files/areas changed.
- Verification commands run and results.
- What was not run and why.
- Known risks, follow-ups, or manual QA needed.

Do not call work production-ready if critical acceptance criteria are incomplete, tests fail, or verification was skipped without a clear reason.
