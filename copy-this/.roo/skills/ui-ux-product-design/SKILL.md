# UI/UX Product Design

Use when user flows, pages, dashboards, forms, or product states are involved.

Think through:
- Primary user intent.
- Jobs-to-be-done.
- Happy path and edge states.
- Empty/loading/error/success states.
- Accessibility and keyboard flow.
- Mobile and desktop behavior.
- Microcopy and affordances.
