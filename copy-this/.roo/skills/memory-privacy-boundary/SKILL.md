---
name: memory-privacy-boundary
description: Apply privacy and safety boundaries before using persistent memory.
---

Use before saving or summarizing memory.

Do not remember:
- API keys, tokens, passwords, private keys
- customer/user personal data
- raw credentials, cookies, JWTs, session IDs
- private business data not meant for persistent agent storage
- temporary guesses or unverified claims

Prefer remembering patterns and decisions rather than raw data.

When unsure, ask the user before saving.
